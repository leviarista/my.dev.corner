# My Dev Corner

My Dev Corner is a virtual space for developers with useful widgets for everyday use.

## Widgets

-   Text
-   Sticker
-   Analog Clock
-   DevTunes FM Radio
-   X-Team Radio
-   freeCodeCamp Radio
-   devMode.fm Podcasts
-   dev.to Posts
-   They Said So Quotes

Also you can switch between backgrounds (color, image or video)
